# G3 Architects
G3 Architects is a modern and responsive architecture landing page built using HTML, CSS, and JavaScript. The project showcases a professional architectural firm website layout with clean UI design and smooth responsiveness.

🚀 Live Demo
🔗 https://g3-architects-three.vercel.app/

📌 Features
Fully responsive design (Mobile, Tablet, Desktop)
Modern architecture-themed UI
Hero section with strong visual impact
Project and feature showcase sections
Interactive elements using JavaScript
Clean navigation and structured layout
Optimized for performance and usability
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
Vercel / Netlify Deployment

⚙️ Setup & Usage
Clone the repository:
git clone https://github.com/istewakhassantewak/g3-architects.git
Navigate to the project folder:
cd g3-architects
Open index.html in your browser.

No frameworks required — runs as a pure static website.
